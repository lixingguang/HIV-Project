Contact networks in FAVITES format

Naming Scheme: cn_MM_n#_e#
  * cn = "contact network"
  * MM = random graph model
    * ba = Barabasi-Albert (BA)
    * er = Erdos-Renyi (ER)
  * n# = # nodes in contact network
  * e# = # edges per node on average


Files:
  * cn_ba_n100000_e1.txt
    * BA model
    * n = 100000 nodes
    * e = 1 edge per node
      * Used m = 1 to get this

  * cn_ba_n100000_e5.txt
    * BA model
    * n = 100000 nodes
    * e = 5 edges per node
      * Used m = 5 to get this

  * cn_ba_n100000_e10.txt
    * BA model
    * n = 100000 nodes
    * e = 10 edges per node
      * Used m = 10 to get this

  * cn_er_n100000_e1.txt
    * ER model
    * n = 100000 nodes
    * e = 1 edge per node
      * Used p = 0.00002 to get this

  * cn_er_n100000_e5.txt
    * ER model
    * n = 100000 nodes
    * e = 5 edges per node
      * Used p = 0.0001 to get this

  * cn_er_n100000_e10.txt
    * ER model
    * n = 100000 nodes
    * e = 10 edges per node
      * Used p = 0.0002 to get this
