Transmission networks in FAVITES format (using contact networks in ../contact_networks)

Naming Scheme: tn_MM_EE_n#_e#_i#
  * cn = "contact network"
  * MM = random graph model
    * ba = Barabasi-Albert (BA)
    * er = Erdos-Renyi (ER)
  * EE = epidemiological model
    * si = Susceptible-Infected (SI)
  * n# = # nodes in contact network
  * e# = # edges per node on average
  * i# = # seed nodes
